# STONKS | Smart, Minimalist Portfolio Intelligence

**STONKS** cuts through the noise of modern finance apps to put powerful stock analytics into a single, distraction-free interface. The platform allows users to seamlessly build personalized portfolios, curate watchlists, and analyze market trends via interactive technical charts and aggregated financial disclosures.

### Key Features

- **Streamlined Tracking:** Centralized portfolio performance, key metrics, and recent financial news.
- **Predictive AI Intelligence:** Powered by in-house **PatchTST models**, delivering automated, daily BUY/HOLD/SELL recommendations directly alongside your assets.
- **Modern Architecture:** Engineered for secure, cross-device synchronization and high-performance data rendering.
